# Downloadable reference gliders

These are useful real-world comparison designs. Check each model page for the current files, print instructions, attribution, and remix restrictions before downloading or redistributing it.

- **Stable Flyer VI - Small Toy Glider** (MakerWorld): one-piece hand-thrown PLA glider, adjustable elevator, approximately 6 g for the light version. https://makerworld.com/en/models/1097660-stable-flyer-vi-small-toy-glider
- **Free-flight glider "WingStar"** (MakerWorld): compact modular free-flight glider with adjustable wing position. https://makerworld.com/es/models/912363-free-flight-glider-model-wingstar
- **Lightweight Gliding Plane** (MakerWorld): small quick-print glider advertised around 5 g. https://makerworld.com/en/models/1228832-lightweight-gliding-plane
- **Large Chuck Glider Remixed** (MakerWorld): larger PLA+ glider with source/STEP files and documented nose ballast; this one is larger than the 250 mm target and is useful mainly as a mass/CG reference. https://makerworld.com/en/models/670782-large-chuck-glider
- **Glider by Busisapro2657** (Thingiverse): CC BY-SA reference aircraft. https://www.thingiverse.com/thing:7000010

No third-party mesh is included in this repository. The generated `out/presets/*.stl` files are native planes-rl baselines and fit the 250 mm volume.
